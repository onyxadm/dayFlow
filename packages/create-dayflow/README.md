# create-dayflow

Interactive setup wizard for DayFlow — the universal calendar component library for React, Vue, Svelte, and Angular.

## Usage

You can use this CLI to quickly set up DayFlow in your project. It will help you install the necessary adapter for your framework, pick plugins, and even configure Tailwind CSS for you.

Run the following command in your project root:

```bash
npx create-dayflow
```

Or with your preferred package manager:

```bash
pnpm dlx create-dayflow
yarn dlx create-dayflow
bun x create-dayflow
```

## Features

- **Framework Detection**: Detects your framework and suggests the correct `@dayflow` adapter.
- **Plugin Selection**: Choose from Drag & Drop, Sidebar, Keyboard Shortcuts, and Localization.
- **Tailwind Integration**: Automatically configures Tailwind v4 scanning for DayFlow components.
- **Package Manager Detection**: Identifies your lockfile and uses the appropriate package manager.
