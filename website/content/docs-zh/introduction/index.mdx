# 快速上手

DayFlow 是一套围绕"即插即用的日历体验"打造的现代日历组件库，支持 React, Vue, Svelte 和 Angular。

### 安装 DayFlow

选择你的框架和首选包管理器以获取安装命令。

<FrameworkInstall />

### 引入 DayFlow 样式

还需要把 DayFlow 的基础样式拉进来，通常在应用入口处加一行即可。

```tsx
// App.tsx or layout.tsx
import '@dayflow/core/dist/styles.css';
```

也可以在全局 CSS 文件中导入：

```css
/* src/index.css */
@import '@dayflow/core/dist/styles.css';
```

**使用 Tailwind CSS？** 改为导入 `styles.components.css`，避免 CSS Reset 与你现有样式冲突：

```tsx
// Tailwind CSS 项目专用 — 不含 CSS Reset，不影响宿主样式
import '@dayflow/core/dist/styles.components.css';
```

    import {

      DayFlowCalendar,
      createDayView,
      createWeekView,
      createMonthView,
      createEventsPlugin,
    } from '@dayflow/react';
    import { createDragPlugin } from '@dayflow/plugin-drag';
    import { createEvent, createAllDayEvent } from '@dayflow/core';
    import '@dayflow/core/dist/styles.css';

    function App() {
      const calendar = useCalendarApp({
        views: [createDayView(), createWeekView(), createMonthView()],
        plugins: [createDragPlugin(), createEventsPlugin()],
        events: [

            id: '1',
            title: '团队会议',

            end: new Date(2025, 10, 15, 11, 0),
          }),
          createAllDayEvent('2', '全天会议', new Date(2025, 10, 20)),
        ],
        initialDate: new Date(),
      });

      return (
        <DayFlowCalendar calendar={calendar} />
      );
    }

### 自定义样式

默认情况下，日历会自动撑满其容器的宽度。你可以通过 CSS 自定义日历的高度或其他样式。例如，设置响应式高度：

```css
.df-calendar-container {
  --df-calendar-height: 760px !important;
  height: var(--df-calendar-height, 760px) !important;
}

@media (max-width: 768px) {
  .df-calendar-container {
    --df-calendar-height: 550px !important;
  }
}
```

> 提醒：DayFlow 底层使用 [Temporal API](/docs-zh/temporal-migration) 处理日期/时间，helper 会替你做好转换，无需直接接触 Temporal。
